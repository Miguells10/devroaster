# DevRoaster 🔥

> "Paste your code. Get roasted."

DevRoaster is the most brutal code review tool in the market. Stop being sensitive about your "clean code" and let the community (and our merciless AI) highlight every bad practice, tech debt, and questionable logic you've ever written.

## 🚀 Features

- **Brutal Roast Mode**: No sugar-coating. Just cold, hard truth about your variable naming and nested loops.
- **Live Leaderboard**: See who's writing the worst code on the internet. Ranked by shame.
- **Syntax Highlighting**: Even garbage code deserves to look good. Powered by Shiki with the Vesper theme.
- **Modern Tech Stack**: Built with Next.js 15 and Tailwind CSS v4 for that sleek, high-performance brutalist feel.

## 🛠️ How it Works

1. **Paste**: Drop your code snippet in our custom editor.
2. **Roast**: Hit the button and enable "Maximum Sarcasm" for the full experience.
3. **Learn (Maybe)**: Get a score and see exactly where you failed.

## 🌈 Design System

Built with a custom-crafted design system focusing on:
- **JetBrains Mono** for that technical, code-first vibe.
- **Brutalist Aesthetics**: High contrast, sharp borders, and vibrant accents.
- **Accessibility**: Built on top of Base UI primitives.
